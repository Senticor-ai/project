# BucketBadge

Color-coded type indicator showing which GTD bucket an item belongs to.
Each bucket has a distinct color token and [Material Symbol](https://fonts.google.com/icons) icon.

## Bucket Reference

| Bucket        | Color Token             | Icon                   |
| ------------- | ----------------------- | ---------------------- |
| Inbox         | `--color-gtd-inbox`     | `inbox`                |
| Next Actions  | `--color-gtd-next`      | `bolt`                 |
| Projects      | `--color-gtd-project`   | `folder`               |
| Waiting For   | `--color-gtd-waiting`   | `schedule`             |
| Someday/Maybe | `--color-gtd-someday`   | `cloud`                |
| Calendar      | `--color-gtd-calendar`  | `calendar_month`       |
| Reference     | `--color-gtd-reference` | `book`                 |
| Focus         | `--color-gtd-focus`     | `center_focus_strong`  |

Colors use a 15% tint background with full-saturation text, following the
[Paperclip color philosophy](?path=/docs/design-design-philosophy--docs).
