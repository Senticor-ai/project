# BucketBadge

Color-coded type indicator showing which bucket an item belongs to.
Each bucket has a distinct color token and [Material Symbol](https://fonts.google.com/icons) icon.

## Bucket Reference

| Bucket    | Color Token             | Icon                  |
| --------- | ----------------------- | --------------------- |
| Inbox     | `--color-app-inbox`     | `inbox`               |
| Next      | `--color-app-next`      | `bolt`                |
| Projects  | `--color-app-project`   | `folder`              |
| Waiting   | `--color-app-waiting`   | `schedule`            |
| Later     | `--color-app-someday`   | `cloud`               |
| Calendar  | `--color-app-calendar`  | `calendar_month`      |
| Reference | `--color-app-reference` | `book`                |
| Focus     | `--color-app-focus`     | `center_focus_strong` |

Colors use a 15% tint background with full-saturation text, following the
[Paperclip color philosophy](?path=/docs/design-design-philosophy--docs).
