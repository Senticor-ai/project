## Design Notes

- Each event is a compact ObjectCard (padding 4px 8px, no subtitle, no shadow)
- Drag to reschedule (snap to 15-min grid)
- All-day events as colored banners at top of day column
- Recurring events show a subtle cycle icon
- Today column highlighted with blueprint background tint
