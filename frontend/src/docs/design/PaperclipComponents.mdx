# Paperclip Component Specifications

Every UI element in Terminandoyo is built from the **Paperclip design language**:
physical paper metaphors, attach/detach interactions, hover-reveal inspectors,
and structure-revealing visuals.

This document specifies each component's visual design, interaction patterns,
and composition rules.

---

## Primitives (Built)

### ObjectCard

The **universal atom** â€” every GTD item renders as an ObjectCard.

| Property         | Visual                                                             |
| ---------------- | ------------------------------------------------------------------ |
| Resting          | Paper surface, subtle shadow (`--shadow-card`)                     |
| Hover            | Lifts 2px, shadow deepens (`--shadow-card-hover`), 150ms ease-snap |
| Selected         | Blueprint border-left (3px), elevated shadow                       |
| Focused          | Star icon (industrial orange) in top-right corner                  |
| Needs enrichment | Sparkle badge pulsing at 50% opacity                               |
| Completed        | Strikethrough title, 0.5 opacity, slide-out after 1s               |

**Composition slots**:

- **Top-left**: BucketBadge (color-coded type indicator)
- **Top-right**: ConfidenceBadge (enrichment state)
- **Bottom**: AttachmentChip row (typed references)
- **Body**: Title, subtitle, context chips, time/energy indicators

### ConfidenceBadge

From [FRBR evidence-first pattern](?path=/docs/engineering-frbr-architecture--docs).

## Composition Rules

### Card Sizes

| Context                           | Card Size                               | Shadow                |
| --------------------------------- | --------------------------------------- | --------------------- |
| In a list (ActionList, InboxList) | Standard (padding: 12px 16px)           | `--shadow-card`       |
| In focus list                     | Hero (padding: 16px 20px)               | `--shadow-card-hover` |
| In calendar                       | Compact (padding: 4px 8px, no subtitle) | None                  |
| In project tree                   | Standard + indent level                 | `--shadow-card`       |
